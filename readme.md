todo
====

Simple commandline todo list manager


Install
-------

```
wget -qO- https://raw.githubusercontent.com/dariubs/todo/master/install.sh | bash
```


How to use
----------

- Add new todo

```bash
todo add
```

- List of tasks

```bash
todo list
```

- Change task status

```bash
todo change id

# or
todo doing id
todo done id
todo archive id
```
